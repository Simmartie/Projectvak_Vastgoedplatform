🔧 Technologie Stack

  • Frontend: Next.js 16.0.3, React 19.2.0, TypeScript, Tailwind CSS 4.1.9
  • UI Components: Radix UI Primitives
  • AI: AI SDK 5.0.93, @ai-sdk/openai 2.0.68
  • Deployment: Vercel Platform met Analytics
  • Data: Mock data in TypeScript (toekomst: PostgreSQL/MongoDB)
